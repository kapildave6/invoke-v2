# WM-4 — Create Window Layout Command (capture-current) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Let users snapshot the current multi-window arrangement as a named "Window Layout" command that re-positions all those apps' windows when run.

**Architecture:** (1) `AppSettings` persists `WindowLayout`s. (2) `WindowEnumerator` (AX-native, WM-2) gains pure `boundsToFractions`/`fractionsToAXBounds` (inverses), `captureCurrentWindows()` (reuses `windowsOnActiveDesktop`), and `applyLayout(items)`. (3) `AppController` adds a "Create Window Layout Command" builder (a native form: a checkbox per current window + a name) + surfaces stored layouts as bindable commands + ⌘K Edit/Delete.

**Tech Stack:** Swift (`InvokeServices` engine + `AppSettings` store + `AppController` flow). Pure logic via standalone `swiftc`; AX via build + relaunch + human.

## Global Constraints
- Faithful parity; world-class UX; commit on `main`.
- **Build/relaunch (MANDATORY — never ad-hoc):** `INVOKE_SIGN_IDENTITY="Invoke Dev" bash scripts/build-app.sh` → `ditto apps/macos/.build/Invoke.app /Applications/Invoke.app` → `pkill -9 -f "Invoke.app/Contents/MacOS/invoke"` → `open /Applications/Invoke.app` → verify ONE PID. (Ad-hoc breaks the Accessibility grant.)
- No Xcode/XCTest → pure logic via standalone `swiftc`; AX via build + relaunch + human. **Ignore SourceKit false-positives when `swift build` prints `Build complete!`** (BrowserDriver/AppleScriptRunner/WindowEnumerator/'let' binding/etc.).
- WM-4 engine lives in `WindowEnumerator` (`InvokeServices`, AX-native coords — NO Cocoa flip; imports only AppKit + ApplicationServices). The store is `InvokeShell`/`AppSettings`. The host maps + builds the form.

## Coordinate note
Layout fractions are **top-left origin** relative to the window's screen **`visibleFrame`**, in AX coords (same space WM-2's `WindowInfo.x/y` use). `boundsToFractions` and `fractionsToAXBounds` are exact inverses → round-trip pinned by a test. (WM-4 does NOT reuse `WindowManager.rectFromFractions`, which is Cocoa-flip for WM-1/WM-3.)

```swift
    /// Window AX bounds → top-left fractions of the screen's visible frame (AX coords). Inverse of fractionsToAXBounds.
    public static func boundsToFractions(winAX w: CGRect, screenVisibleAX sv: CGRect) -> (fx: Double, fy: Double, fw: Double, fh: Double) {
        guard sv.width > 0, sv.height > 0 else { return (0, 0, 1, 1) }
        return (Double((w.minX - sv.minX) / sv.width), Double((w.minY - sv.minY) / sv.height),
                Double(w.width / sv.width), Double(w.height / sv.height))
    }
    /// Top-left fractions → window AX bounds within the screen's visible frame. Inverse of boundsToFractions.
    public static func fractionsToAXBounds(fx: Double, fy: Double, fw: Double, fh: Double, screenVisibleAX sv: CGRect) -> CGRect {
        CGRect(x: sv.minX + CGFloat(fx) * sv.width, y: sv.minY + CGFloat(fy) * sv.height,
               width: CGFloat(fw) * sv.width, height: CGFloat(fh) * sv.height)
    }
    private func primaryFullHeight() -> CGFloat {
        (NSScreen.screens.first { $0.frame.origin == .zero } ?? NSScreen.main)?.frame.height ?? 0
    }
    private func visibleAX(forCenter c: CGPoint) -> CGRect? {
        let ph = primaryFullHeight()
        let screen = NSScreen.screens.first { Self.axRect(cocoaFrame: $0.frame, primaryHeight: ph).contains(c) } ?? NSScreen.main
        guard let s = screen else { return nil }
        return Self.axRect(cocoaFrame: s.visibleFrame, primaryHeight: ph)
    }
    /// Snapshot the current on-screen windows as layout items (fractions of each window's screen visibleFrame).
    public func captureCurrentWindows() -> [LayoutCaptureItem] {
        guard hasAccessibility else { return [] }
        var out: [LayoutCaptureItem] = []
        for w in windowsOnActiveDesktop() {
            guard let bid = w.bundleId else { continue }
            let center = CGPoint(x: w.x + w.width / 2, y: w.y + w.height / 2)
            guard let sv = visibleAX(forCenter: center) else { continue }
            let f = Self.boundsToFractions(winAX: CGRect(x: w.x, y: w.y, width: w.width, height: w.height), screenVisibleAX: sv)
            out.append(LayoutCaptureItem(bundleId: bid, appName: w.appName ?? bid, fx: f.fx, fy: f.fy, fw: f.fw, fh: f.fh))
        }
        return out
    }
    /// Position each item's app main window to its fractional bounds. Returns #positioned (skips non-running apps).
    @discardableResult
    public func applyLayout(_ items: [(bundleId: String, fx: Double, fy: Double, fw: Double, fh: Double)]) -> Int {
        guard hasAccessibility else { return 0 }
        var n = 0
        for item in items {
            guard let app = NSWorkspace.shared.runningApplications.first(where: { $0.bundleIdentifier == item.bundleId }) else { continue }
            let axApp = AXUIElementCreateApplication(app.processIdentifier)
            var ref: CFTypeRef?
            var win: AXUIElement?
            if AXUIElementCopyAttributeValue(axApp, kAXMainWindowAttribute as CFString, &ref) == .success, let r = ref, CFGetTypeID(r) == AXUIElementGetTypeID() {
                win = (r as! AXUIElement)
            } else {
                var wins: CFTypeRef?
                if AXUIElementCopyAttributeValue(axApp, kAXWindowsAttribute as CFString, &wins) == .success, let arr = wins as? [AXUIElement], let first = arr.first { win = first }
            }
            guard let window = win else { continue }
            // window center → its screen visibleFrame (AX) → target AX bounds
            guard let pos = axDouble(window, kAXPositionAttribute as String, .cgPoint),
                  let sz = axDouble(window, kAXSizeAttribute as String, .cgSize),
                  let sv = visibleAX(forCenter: CGPoint(x: pos.0 + sz.0 / 2, y: pos.1 + sz.1 / 2)) else { continue }
            let target = Self.fractionsToAXBounds(fx: item.fx, fy: item.fy, fw: item.fw, fh: item.fh, screenVisibleAX: sv)
            var newSize = CGSize(width: target.width, height: target.height)
            var newPos = CGPoint(x: target.minX, y: target.minY)
            if let v = AXValueCreate(.cgSize, &newSize) { AXUIElementSetAttributeValue(window, kAXSizeAttribute as CFString, v) }
            if let v = AXValueCreate(.cgPoint, &newPos) { AXUIElementSetAttributeValue(window, kAXPositionAttribute as CFString, v) }
            if let v = AXValueCreate(.cgSize, &newSize) { AXUIElementSetAttributeValue(window, kAXSizeAttribute as CFString, v) }
            n += 1
        }
        return n
    }
```
(`axDouble` is the existing private WM-2 helper. If `primaryFullHeight()` already exists privately in WindowEnumerator, reuse it instead of re-adding.)
